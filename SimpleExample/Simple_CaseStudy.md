
# Simple Case Study: Single-Visit Persistence in a Nonlinear Platoon

This document details the original, simple computational example of Path-Complete Closure Certificates (PC-CC) applied to a nonlinear discrete-time switched system, as presented in the main manuscript. 

In this scenario, the initial and finite-visit regions are defined such that the system trajectories leave the finite-visit set after a **single step** and do not return. Consequently, the well-foundedness condition (PC-CC3) is satisfied vacuously, making this an introductory illustrative example of automated graph-indexed certificate synthesis via Sum-of-Squares (SOS) relaxations.

## 1. System Dynamics

We consider a two-vehicle platoon system where the state vector $x = [x_1, x_2]^\top$ consists of the follower ($x_1$) and leader ($x_2$) velocities. The system switches arbitrarily between two operational modes: Mode 1 (normal communication) and Mode 2 (communication breakdown). 

The discrete-time polynomial dynamics are given by:

$$
f_1(x) = 
\begin{bmatrix}
0.01 x_2 + 0.9 x_1 - 0.02 x_1^2 \\
2.0 + 0.8 x_2 - 0.04 x_2^2
\end{bmatrix}
$$

$$
f_2(x) = 
\begin{bmatrix}
0.9 x_1 - 0.02 x_1^2 \\
2.0 + 0.8 x_2 - 0.04 x_2^2
\end{bmatrix}
$$

## 2. State Space and Target Sets

The system is bounded within a compact state space $\mathcal{X}$, where velocities are constrained between $0$ and $6$, and the leader is always moving at least as fast as the follower ($x_2 \ge x_1$):

$$
\mathcal{X} = \{x \in \mathbb{R}^2 : 0 \le x_1 \le 6, \; 0 \le x_2 \le 6, \; x_2 - x_1 \ge 0\}
$$

Let $g(x) = x_2 - x_1$ denote the inter-vehicle gap. The initial state set $\mathcal{X}_0$ and the finite-visit set $\mathcal{X}_{\mathrm{vf}}$ are defined by tight gap bounds:

$$
\mathcal{X}_0 = \{x \in \mathcal{X} : g(x) \le 0.3\}
$$

$$
\mathcal{X}_{\mathrm{vf}} = \{x \in \mathcal{X} : g(x) \le 0.2\}
$$

Under these specific bounds, the nonlinear dynamics aggressively separate the vehicles, causing the trajectories to escape $\mathcal{X}_{\mathrm{vf}}$ completely after a single time step.

## 3. PC-CC Synthesis and Results

For both graph configurations, the SOS framework rapidly synthesized a low-degree **Degree 2** Path-Complete Closure Certificate ($C_{p,q}(x,y) \in \Sigma[x,y]_2$) using standard fixed scalar multipliers ($\tau = 1$). 

The extraction of the degree-2 certificates demonstrates that:
1. The trajectories visit $\mathcal{X}_{\mathrm{vf}}$ only finitely many times (exactly once).
2. The certificates evaluate to strict negative values inside $\mathcal{X}_{\mathrm{vf}}$, rendering the strict decrease condition (PC-CC3) **vacuously true**, as described in the Algorithmic Interpretation remark of the manuscript.

*(The full degree-2 polynomial coefficients are generated by the `S_pccc_onenode.jl` and `S_pccc_twonode.jl` scripts in this repository).*
